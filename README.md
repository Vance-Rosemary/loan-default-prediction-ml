# Loan Default Prediction Using Machine Learning

## Project Overview

This project uses machine learning techniques to predict whether a borrower is likely to default on a loan using financial and demographic data.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Machine Learning Models

* Logistic Regression
* Random Forest Classifier

## Key Findings

* Accuracy alone was misleading due to class imbalance.
* Logistic Regression detected more defaulters.
* Loan amount, credit score, and interest rate strongly influenced prediction outcomes.

## Conclusion

The project demonstrated how machine learning can support financial risk assessment and borrower default prediction.
